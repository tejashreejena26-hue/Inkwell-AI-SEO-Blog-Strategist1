# Inkwell AI – SEO Blog Strategist

Inkwell AI is a web application that helps users generate SEO-optimized blog content using artificial intelligence. The tool assists content creators, marketers, and bloggers in quickly creating structured blog outlines and optimized articles.

## 🚀 Features

- AI-powered SEO blog generation
- Keyword-based blog content suggestions
- Clean and modern user interface
- Fast performance using Vite
- Built with React and TypeScript
- Easy to extend for future AI integrations

## 🛠 Tech Stack

Frontend:
- React
- TypeScript
- Vite
- CSS

Development Tools:
- Node.js
- npm

## ⚙️ Installation

1. Clone the repository

```
git clone https://github.com/your-username/Inkwell-AI-SEO-Blog-Strategist1.git
```

2. Navigate into the project folder

```
cd Inkwell-AI-SEO-Blog-Strategist1
```

3. Install dependencies

```
npm install
```

4. Start the development server

```
npm run dev
```

The application will start on a local development server.

## 💡 How It Works

1. User enters a blog topic or keyword.
2. The system processes the input.
3. AI generates structured blog content optimized for SEO.
4. The user can edit and use the generated content.
